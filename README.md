# Bag sales

A React + Vite project built with TypeScript.

## Tech Stack

- **React + Vite**
- **TypeScript**
- **CSS**
- **ESLint**
- **Prettier**

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd bag sales

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```
